# Chat Bot - AI Assistant

A responsive web-based chat application that integrates with the Gemini AI API to provide intelligent responses to user queries.

## Features

- **AI-Powered Responses**: Uses Google's Gemini API for natural language processing
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Dark/Light Mode**: Toggle between dark and light themes
- **Clear Chat**: Easily clear conversation history with one click
- **Markdown Support**: Supports bold and italic text formatting in responses
- **Real-time Typing Indicator**: Shows when the AI is generating a response

## Setup Instructions

1. Clone or download this repository
2. Open `index.html` in a web browser
3. Start chatting with the AI assistant!

## API Configuration

The application uses the Gemini API with the following configuration:
- Model: `gemini-2.0-flash`
- API Key: `key`

## Usage

1. Type your message in the input field at the bottom
2. Press Enter or click the send button to submit
3. The AI will respond in the chat window
4. Use the trash icon to clear the chat history
5. Use the theme toggle to switch between dark and light modes

## Special Commands

The chatbot has special responses for certain queries:
- "Who are you?" - Introduces itself as Chat Bot
- "Who developed you?" - Lists the developer names

## Responsive Design

The application is fully responsive and works on:
- Desktop browsers (Chrome, Firefox, Safari, Edge)
- Mobile devices (iOS Safari, Android Chrome)
- Tablets
